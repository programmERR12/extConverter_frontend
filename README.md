# 🚀 ExtConverter Frontend

A modern and responsive frontend for a File Converter web application that allows users to convert files like PDF ↔ Word easily using a backend API.

---

## ✨ Features

- 📄 PDF to Word conversion
- 📄 Word to PDF conversion
- ⚡ Fast and responsive UI
- 📱 Mobile friendly design
- 🔄 Real-time file upload
- ⬇️ Instant file download after conversion
- 🎨 Clean and minimal UI

---

## 🛠️ Tech Stack

- ⚛️ React.js
- ⚡ Vite
- 🎨 CSS / Tailwind (if used)
- 🔗 REST API integration
- 📦 Axios / Fetch API

---
## 🚀 Getting Started

Follow these steps to run the frontend locally on your machine.

---

### 📌 Prerequisites

Make sure you have installed:

- Node.js (v16 or higher recommended)
- npm or yarn

---

### 📥 1. Clone the Repository

```bash
# 1. Clone repo
git clone https://github.com/programmERR12/extConverter_frontend.git

# 2. Go to project folder
cd extConverter_frontend

# 3. Install dependencies
npm install

# 4. Start dev server
npm run dev
